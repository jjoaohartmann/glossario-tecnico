background-color

Serve para mudar a cor de fundo dos elementos.

background-color Cor de fundo
background-color: #FFF body {
background-color: #fff;
}

Alinhar ao centro

Esse é um possível jeito para alinhar os elementos no centro.

align-items Alinhar o conteúdo
align-items: center; .cards {
margin: auto;
align-items: center;
}

margem

definições para aplicar a margem nos elementos.

Margin Margem dos elementos
.cards {
margin: auto;
margin-top: (margem topo);
margin-bottom: (margem abaixo);
margin-left: (margem esquerda);
margin-right: (margem direita);
}

Ajuste de tamanho

Para ajustar os elemento é usado as tags width e heigth para que consiga aumentar e diminuir o tamanho tanto do fundo quanto de uma imagem.

heigth e width Ajuste de tamanho de elementos
.cards{
heigth:90px/50%; (altura do elemento)
width:90px/50% (Largura do elemento);

também exite um codigo que ele se ajusta de acordo a quantidade de elementos dentro:

width: fit-content;
heigth: fit-content;
}

Sombreamento de uma caixa

Essa propriedade do Css serve para que algum elemento que ficaria melhor com um Sombreamento ou uma animação que ficaria melho quando passase o cursor por cima.

Box-shadow Sombra em uma caixa/elemento
.cards {
box-shadow: 0 2px 5px rgba(0, 0, 0, 0.05);
}

Script

Essa tag é utilizada quando é necessário vincular o arquivo HTML ao JavaScript para que haja interações dentro da página.

script Vincular JS ao HTML
<script>

Após inserir a tag, é necessário vincular o arquivo colocando src="[nome-do-arquivo.js]" dentro da tag de abertura.

</script>

Strong

Essa tag deixa em negrito as palavras que ficam entre a abertura e o fechamento da tag, sendo mais utilizada quando se quer destacar palavras específicas dentro de um texto.

strong Deixa uma palavra em negrito.
<strong>

E aqui você coloca a palavra ou o texto escolhido para o destaque.

</strong>

Input

Serve para criar campos de entrada de dados em formulários web, permitindo que os usuários interajam com a página e enviem informações.

input Caixa para inserir dados.
<input>

A partir disso, aparecerá uma caixa. Para fazer com que apareça um texto no fundo dessa caixa, basta usar o atributo placeholder.

Button

É usada para criar um botão clicável em uma página da web.

button Cria um botão para interagir no site.
<button>

Após a inserção da tag button, pode ser colocado texto ou até outras tags dentro, como p ou a.

a

É usada para criar hiperlinks, ou seja, links que permitem aos usuários navegar entre diferentes páginas da web, seções dentro da mesma página ou até mesmo para arquivos e endereços de e-mail.

a Cria um link para navegação.
<a href="[aqui é inserido o link]"> </a>
