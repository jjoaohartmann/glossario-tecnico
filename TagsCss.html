background-color

Serve para mudar a cor de fundo dos elementos.

background-color Cor de fundo
background-color: #FFF body {
background-color: #fff;
}

Alinhar ao centro

Esse é um possível jeito para alinhar os elementos no centro.

align-items Alinhar o conteúdo
align-items: center; .cards {
margin: auto;
align-items: center;
}

margem

Definições para aplicar a margem nos elementos.

Margin Margem dos elementos
.cards {
margin: auto;
margin-top: (margem topo);
margin-bottom: (margem abaixo);
margin-left: (margem esquerda);
margin-right: (margem direita);
}

Ajuste de tamanho

Para ajustar os elementos é usado as tags width e height para que consiga aumentar e diminuir o tamanho tanto do fundo quanto de uma imagem.

height e width Ajuste de tamanho de elementos
.cards{
height:90px/50%; (altura do elemento)
width:90px/50% (largura do elemento);

Também existe um código que se ajusta de acordo com a quantidade de elementos dentro:

width: fit-content;
height: fit-content;
}

Sombreamento de uma caixa

Essa propriedade do CSS serve para que algum elemento que ficaria melhor com um sombreamento ou uma animação que ficaria melhor quando passasse o cursor por cima.

Box-shadow Sombra em uma caixa/elemento
.cards {
box-shadow: 0 2px 5px rgba(0, 0, 0, 0.05);
}

Barra de rolagem

Para editar a forma como funciona a barra de rolagem das páginas.

Overflow Define estilos de barra de rolagem
.cards {
overflow-x(serve para a barra de rolagem Horizontal);
overflow-y(serve para a barra de rolagem Vertical);
}

E após a seleção de qual barra gostaria de por algum atributo é só colocar dois pontos e escolher um:

Hidden; (esconde a barra) Scroll; (Aparece quando o conteúdo excede o espaço) auto; (Adiciona automaticamente a barra) visible: (dimensões de um elemento exibidas normalmente, sem ser cortado ou ter barras de rolagem adicionadas)

Espaçamento Interno do elemento

Como a propriedade "Margin" que define um espaçamento externo dos elementos, também existe o padding que define um espaçamento interno dos elementos.

padding Espaçamento interno
.cards {
padding:10px;
}

Como exemplo esse botão que tem um padding de 10px

Display

Ela controla o tipo de caixa que o elemento usa e como ele interage com outros elementos ao seu redor.

Display controla o tipo de caixa e interação
.cards {
display: block; Não permite que outros elementos fiquem na mesma linha ao seu lado.
display: inline; Faz com que um elemento seja exibido na mesma linha que outros elementos, sem quebrar a linha antes ou depois dele.
display: flex; Serve para tornar um display mais flexível e fácil de ajustar.
display: grid; Para criar um layout em grade, permitindo organizar elementos filhos em linhas e colunas de forma bidimensional.
}

hover

É uma pseudo-classe CSS que reage à interação do usuário com o mouse. É usado para aplicar estilos específicos a um elemento quando o cursor do mouse está sobre ele.

hover Animação feita em um elemento quando passa o cursor
.animacao:hover {
background-color: aliceblue;
color: red;
}

Text-transform

A propriedade "text-transform" pode definir como um texto deve ser capitalizado. Alguns exemplos são o "uppercase", que deixa todas as letras maiúsculas; "lowercase", que deixa todas as letras minúsculas e o "none" que mantém o texto como está.

text-transform – Utilizada para definir como o texto deve ser exibido (letras maiúsculas apenas, definir a letra inicial)
text-transform: uppercase
