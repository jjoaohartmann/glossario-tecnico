Tags Html

  • INÍCIO

Header

Serve para criar cabeçalhos no início da página.

header Cabeçalho

<header> Aqui podem ser colocadas imagens, títulos ou até mesmo uma barra de navegação feita com a <ul>. </header>

Class

Serve para classificar as tags, permitindo uma formatação mais fácil.

class Classificação
<div class="container">

Após isso, é só colocar os elementos que quiser aqui dentro e personalizar pelo CSS da forma que preferir usando ".container".

</div>

UL

Serve para fazer listas desordenadas, onde aparecem pontos na frente.

ul Lista desordenada.
<ul>

<li>Conteúdo da lista</li>
<li>Conteúdo da lista</li>

</ul>

Footer

O footer serve para criar rodapés ao final da página, onde geralmente são colocadas informações de contato ou desenvolvimento.

footer Cria um rodapé.
<footer>

Site desenvolvido por [nome da pessoa]

</footer>

Code

É usada para marcar um fragmento de código em uma página HTML, indicando que o texto dentro dela é um trecho de programação ou comando, diferenciando-o do texto comum.

code Marcar um fragmento de código.
<code>

Para criar um parágrafo, use a tag <code>p</code> no HTML.

</code>

Div

É usada para agrupar elementos HTML, servindo como uma “caixa” genérica para organizar o conteúdo da página.

div Cria uma caixa para agrupamento.
<div>

Aqui podem ser colocados diversos elementos, sendo muito utilizada para fazer cards ou divisões dentro da página.

</div>

Script

Essa tag é usada quando é necessário vincular o arquivo HTML ao JavaScript para que haja interações dentro da página.

script Vincular JS ao HTML
<script>

Após inserir a tag, é necessário vincular o arquivo, colocando src="[nome-do-arquivo.js]" dentro da tag de abertura.

</script>

Strong

Essa tag coloca em negrito as palavras que ficam entre a abertura e o fechamento da tag, sendo mais utilizada quando se quer destacar palavras específicas dentro de um texto.

strong Deixa uma palavra em negrito.
<strong>

E aqui você coloca a palavra ou o texto escolhido para o destaque.

</strong>

Input

Serve para criar campos de entrada de dados em formulários web, permitindo que os usuários interajam com a página e enviem informações.

input Caixa para inserir dados.
<input>

A partir disso, aparecerá um tipo de caixa. Para fazer com que apareça um texto no fundo dessa caixa, basta usar o atributo placeholder.

Button

É usada para criar um botão clicável em uma página da web.

button Cria um botão para interagir no site.
<button>

Após a inserção da tag button, pode ser colocado texto ou até outras tags dentro, como p ou a.

</button>

a

É usada para criar hiperlinks, ou seja, links que permitem aos usuários navegar entre diferentes páginas da web, seções dentro da mesma página ou até mesmo para arquivos e endereços de e-mail.

a Serve para hiperlinks.
<a href="[aqui é inserido o link para que as pessoas consigam ir para outras páginas]"> </a>
