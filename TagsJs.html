If Else

É uma estrutura de decisão que, traduzida para o português, seria: Se e Senão. Ou seja, se determinada condição for verdadeira, ocorre uma ação; caso contrário, outra ação será executada

If Else Estrutura de Decisão
let nota = 85;
if(nota >= 90){
console.log("Nota A!");
}else{
console.log("Nota C");
}

Declaração Variável com Let

É usando quando você precisa de uma variável cujo valor pode mudar durante a execução do programa. É ideal para contadores, variáveis de controle em laços ou quando o dado precisa ser atualizado.

Let Declaração de Variável alternavel
let idade = 20;
console.log(idade);
// saída é o 20//

// alterando o valor da variável //
idade = 21;
console.log(idade);
// saída é 21 //

Declaração Variável com Const

É usado quando o valor não deve ser reatribuído. Ele é indicado para valores fixos, constantes matemáticas, configurações ou variáveis que nunca vão mudar ao longo do código.

Const Declaração de Variável fixa
Const cpf = 123.456.789-00;
//CPF é fixo//

console.log("CPF", cpf);

//cpf = "000.111.222-33"//
❌ Isso daria erro, pois const não pode ser reatribuído
